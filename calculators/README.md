## How to use (buyers)

1. Open the `.html` file in a browser to preview.
2. Copy the entire file contents (or the block inside `<body>`).
3. Paste into a WordPress HTML block, Webflow embed, or any page template.
4. Customize colors via CSS variables on the root class (documented in each product README).
