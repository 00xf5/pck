# Premium Agency — Landing Page

**Status:** Ready

**NORTH** — dark, competitive digital-agency template. Open `index.html` or upload the folder.

## Look

- Near-black canvas, acid lime accent, film grain
- Massive Syne display type + Manrope + IBM Plex Mono
- Full-bleed cinematic hero, client marquee, drag-to-scroll work runway
- Giant hover service rows, studio split, contact form

Built to sit next to contemporary agency sites (bold type, dark UI, motion) without a JS framework.
