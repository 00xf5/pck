# PCK UI Kit — Design Tokens (Neutral Luxury)

| Token | Value | Usage |
|-------|-------|--------|
| `pck-ink` | `#0c0a09` | Headings, dark backgrounds |
| `pck-muted` | `#78716c` | Body secondary |
| `pck-surface` | `#fafaf9` | Input backgrounds |
| `pck-cream` | `#f5f0eb` | Page / section warmth |
| `pck-accent` | `#1e3a5f` | Navy CTAs |
| `pck-gold` | `#a6833d` | Editorial accents |
| `pck-line` | `#e7e5e4` | Borders |

**Shadow:** `shadow-pck` → `0 25px 50px -12px rgba(0,0,0,0.1)`  
**Images:** Unsplash placeholders — swap URLs in each HTML file.  
**Buttons:** Prefer `rounded-full` for luxury hospitality feel.
